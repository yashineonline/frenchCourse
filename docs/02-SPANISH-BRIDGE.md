# Spanish → French Bridge (use your advantage)

## Fast wins (Spanish helps)
1. **Cognates** (many shared Latin roots)
   - important / importante
   - nation / nación
   - possible / posible
2. **Verb mindset** (conjugations exist in both languages)
3. **Gender** (often same, not always)

## Biggest traps (false friends)
- actuellement = "currently" (NOT "actualmente" meaning "actually" in English)
- assister à = "to attend" (NOT "assistir" = help)
- sensible = "sensitive" (NOT "sensato")
- librairie = bookstore (NOT "library" = bibliothèque)
- éventuellement = possibly (NOT "eventualmente" same-ish but watch usage)

## Pronunciation differences to fix early
- French **u** (tu) ≠ Spanish u
- French **r** is guttural
- Nasal vowels: en/an, on, in
- Final consonants often silent (parle, petits, vous parlez)

## Grammar differences (high yield)
- French uses **articles** more consistently
- Negation: *ne ... pas* (spoken French often drops *ne*)
- Object pronouns positioning: *je le vois* / *je vais le faire*

## Study hack
When you learn a new French word:
- write the Spanish cognate if it exists
- mark it **(FF)** if it might be a false friend
