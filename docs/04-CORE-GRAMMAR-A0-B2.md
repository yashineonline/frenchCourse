# Core Grammar Ladder (A0 → B2) — only what moves the needle

## A0–A1 (Weeks 1–2): survival structure
- Pronouns: je/tu/il/elle/nous/vous/ils/elles
- Present tense: être, avoir, aller, faire + regular -er
- Negation: ne...pas (and spoken drop of "ne")
- Questions: est-ce que / inversion (basic) / intonation
- Articles: un/une/des, le/la/les, du/de la/des

Use:
- Lawless French by level: https://www.lawlessfrench.com/faq/lessons-by-level/

## A2 (Weeks 3–4): control
- Passé composé (avoir/être) + agreement basics
- Futur proche (aller + infinitive)
- Pronouns: le/la/les, lui/leur, y, en (basics)
- Comparatives: plus/moins/aussi ... que

## B1 (Weeks 5–6): fluency structure
- Imparfait vs passé composé
- Relative pronouns: qui/que/où (core)
- Conditionnel (polite requests + plans)
- Subjunctive: only the “must know” triggers (il faut que, je veux que)

## B2 (Weeks 7–8+): argumentation
- Connectors: cependant, en revanche, puisque, donc, par conséquent
- Hypotheses: si + tenses (realistic patterns)
- Opinion frameworks: à mon avis, il me semble que, je suis convaincu(e) que

## Rule
Do grammar to SUPPORT output (speaking/writing). Don’t “collect rules”.
