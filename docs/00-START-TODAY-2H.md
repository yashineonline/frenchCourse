
# Start Today (2 hours) — Zero to Momentum

## 0) Setup (10 min)
- Create a notebook or a notes file called: **FR-PR-Log**
- Install an SRS (recommended): **Anki**  (ask Yashine about this first if you are unsure)
- Pick a dictionary you like (WordReference works well): https://www.wordreference.com/

## 1) Choose your PR language goal (10 min)
Target for “PR-optimized French”:
- **NCLC 7 in all 4 skills** (Listening, Reading, Writing, Speaking)

Read later: `01-PR-TARGETS-TEF-TCF.md`

## 2) Pronunciation “must-fix early” (20 min)
Do these now (listen + repeat + record yourself):
- French **U** vs **OU** (Spanish speakers often collapse them)
- French **R** (throaty) vs Spanish rolled R
- Nasal vowels: **an/en**, **on**, **in**
- Silent endings + liaison basics

Use:
- Learn French With Alexa (search within channel): https://www.youtube.com/user/learnfrenchwithalexa

## 3) First input session (30 min)
Choose ONE:
- TV5MONDE A1 exercises: https://apprendre.tv5monde.com/fr/exercices/a1-debutant
- RFI easy exercises: https://francaisfacile.rfi.fr/fr/exercices/

Rule: always prefer content with **audio + transcript**.

## 4) First output session (30 min)
### Speaking (15 min)
Record yourself answering:
1. Je me présente (name, work, studies, where you live, goal in Canada)
2. Pourquoi le Canada ?
3. Décrivez votre journée type.

### Writing (15 min)
Write 8–10 lines:
- “Pourquoi je veux améliorer mon français pour le Canada.”

Keep it simple. Accuracy > complexity.

## 5) Micro-vocab starter (20 min)
Create 25 flashcards (FR → ES/EN) from today’s content.
Must include:
- 5 verbs (être, avoir, aller, faire, pouvoir)
- 5 connectors (et, mais, parce que, donc, aussi)
- 5 “Canada life” words (logement, travail, rendez-vous, permis, assurance)
- 10 personal words you actually need
