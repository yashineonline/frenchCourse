# 20 Writing Prompts (TEF/TCF-aligned)

These prompts map to:
- **TCF Canada Expression écrite**:  
  Task 1 (60–120 words), Task 2 (120–150 words), Task 3 (120–180 words, compare 2 viewpoints).  
- **TEF Canada Expression écrite**:  
  Section A (continue an article, min ~80 words), Section B (opinion + justify, min ~200 words).

Time yourself and respect word limits.

---

## How to practice (fast)
1) Write once timed.
2) Underline: verbs, agreements, connectors.
3) Rewrite only the intro + conclusion (highest ROI).

---

# A) [TCF-EE1] Task 1 — Message (60–120 words) (6 prompts)

1. Message à un propriétaire: demande de visite + questions (charges, bail, date).
2. Message à un centre d’examen: demander dates TEF/TCF + inscription + documents.
3. Message à un ami: expliquer votre plan intensif pour apprendre le français.
4. Message à une clinique: demander rendez-vous + disponibilité + carte d’assurance.
5. Message à un employeur: confirmer un entretien + demander modalités.
6. Message à un service client: produit défectueux + demande d’échange.

---

# B) [TCF-EE2] Task 2 — Letter/article/note (120–150 words) (6 prompts)

7. Courrier au syndic/propriétaire: bruit répété → décrire + demander solution.
8. Note à votre équipe: proposer un nouveau système d’organisation (réunions, tâches).
9. Courriel à une école: demander inscription + niveau + objectifs + horaires.
10. Article court: “Vivre au Canada: premiers défis et solutions” (récit + conseils).
11. Lettre à un ami: raconter votre arrivée dans une nouvelle ville + vos impressions.
12. Courriel à un supérieur: demander une formation + expliquer bénéfices pour le travail.

---

# C) [TCF-EE3] Task 3 — Compare 2 viewpoints (120–180 words) (4 prompts)
For each: write a short compare/contrast + your opinion.

13. Doc A: “Télétravail améliore la qualité de vie” vs Doc B: “Télétravail isole”. Donnez votre avis.
14. Doc A: “Transports gratuits” vs Doc B: “Trop cher pour l’État”. Donnez votre avis.
15. Doc A: “Réseaux sociaux utiles” vs Doc B: “Réseaux sociaux dangereux”. Donnez votre avis.
16. Doc A: “Immigration nécessaire” vs Doc B: “Immigration à limiter”. Donnez votre avis.

---

# D) [TEF-EE-A] Section A — Continue an article (min ~80 words) (2 prompts)

17. Article: “Les nouveaux arrivants au Canada…” → continuez: difficultés + 2 solutions concrètes.
18. Article: “Apprendre le français à l’âge adulte…” → continuez: méthode + motivation + exemple.

---

# E) [TEF-EE-B] Section B — Opinion + justify (min ~200 words) (2 prompts)

19. “Le gouvernement devrait encourager davantage l’apprentissage du français au Canada.” Êtes-vous d’accord ? Justifiez.
20. “Le travail à distance est une évolution positive pour la société.” Donnez votre point de vue, arguments, exemple, conclusion.

---

## Mini checklist (before you stop)
- [ ] Word count respected
- [ ] Intro = position claire
- [ ] 2–3 connectors (cependant, donc, en revanche, par conséquent)
- [ ] 1 example concrete
- [ ] Conclusion = synthèse + ouverture
