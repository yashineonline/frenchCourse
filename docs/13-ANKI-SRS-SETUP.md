# SRS + Anki Setup (10 minutes)

## What is SRS?
SRS (Spaced Repetition System) shows you flashcards at the best time to remember them.
It’s one of the highest ROI tools for vocab + useful phrases.

## What is Anki?
Anki is a popular SRS system with:
- desktop app (Windows/Mac/Linux)
- AnkiWeb sync
- Android app (AnkiDroid)
- iOS app (AnkiMobile)

Official site: https://apps.ankiweb.net/ :contentReference[oaicite:0]{index=0}

## Install options

### Android (free): AnkiDroid
Google Play: https://play.google.com/store/apps/details?id=com.ichi2.anki :contentReference[oaicite:1]{index=1}

Steps:
1. Install AnkiDroid
2. Open it → create or sign into AnkiWeb (optional but recommended)
3. Add a deck: “French PR Accelerator”
4. Add 20–40 cards/week from your input sessions

### iPhone/iPad (paid): AnkiMobile
App Store: https://apps.apple.com/ca/app/ankimobile-flashcards/id373493387 :contentReference[oaicite:2]{index=2}

Steps:
1. Buy/install AnkiMobile
2. Enable sync with AnkiWeb
3. Use the same deck across phone + computer

### Desktop (free)
Download from official site: https://apps.ankiweb.net/ :contentReference[oaicite:3]{index=3}

## Daily use (recommended)
- 10–20 minutes/day
- Add 5–10 new cards/day (only from content you actually saw/heard)

## Card format (best for French)
Front: French sentence + audio (optional)  
Back: Meaning (English/Spanish) + 1 grammar note  
Extra: 1 example

Example:
Front: *Je voudrais prendre rendez-vous.*  
Back: “I’d like to make an appointment.” / “Quisiera pedir una cita.”
