# Input Ladder (A0 → B2): listen/read your way up

## A0–A1: super-supported input
- TV5MONDE A1: https://apprendre.tv5monde.com/fr/exercices/a1-debutant

Goal:
- 15 min/day + write 5 sentences you understood.

## A2: controlled authentic input
- TV5MONDE A2: https://apprendre.tv5monde.com/fr/exercices/a2-elementaire
- RFI exercises (all levels): https://francaisfacile.rfi.fr/fr/exercices/

Goal:
- 1 short audio/day + transcript + 10 new words.

## B1: real content, slower
- innerFrench (excellent bridge to real French): https://www.youtube.com/innerFrench
- RFI “Journal en français facile” (with transcript): https://francaisfacile.rfi.fr/fr/podcasts/journal-en-fran%C3%A7ais-facile/

Goal:
- 20–30 min/day + 1 paragraph summary.

## B2: faster, wider topics
- RFI B2 exercises: https://francaisfacile.rfi.fr/fr/exercices/b2/
- Easy French (street interviews): https://www.youtube.com/easyfrench

Goal:
- summarize + react (opinion) out loud for 2 minutes.

## The golden rule
If you understand < 70%, go easier. Volume beats suffering.
