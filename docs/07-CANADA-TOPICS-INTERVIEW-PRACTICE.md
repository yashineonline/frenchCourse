# Canada Topics (PR-relevant) + “Interview-style” prompts

Even when PR uses test scores, you still want French for life + confidence.

## Core topics to master (vocab + speaking)
1) Work & skills (poste, expérience, responsabilités, objectifs)
2) Housing (logement, bail, coloc, quartier, loyer)
3) Healthcare (rendez-vous, assurance, clinique, pharmacie)
4) Banking (compte, carte, virement, facture)
5) Transportation (permis, trajet, abonnement, horaires)
6) Education (diplôme, formation, université)
7) Government/admin (formulaire, document, délai, statut)

## Practice questions (record 2 min each)
- Présentez-vous.
- Pourquoi le Canada ? Pourquoi apprendre le français ?
- Décrivez votre travail / vos études.
- Parlez d’un défi que vous avez surmonté.
- Donnez votre opinion sur le télétravail / l’immigration / l’environnement.
- Expliquez une procédure (ex: prendre un rendez-vous médical).

## Weekly “real-life task”
- Call roleplay: make an appointment, ask for info, negotiate a schedule
- Write: a polite email requesting information about an application
